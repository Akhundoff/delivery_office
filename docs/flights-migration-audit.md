## Notes

- `npx tsc --noEmit` passes with zero errors across all touched files.
- `FlightCountsCell` and `FlightDimensionalWeightCell` use `react-query` with `enabled` flag for lazy fetching with caching, matching old behavior.
- `useTinyFlights` is exported from the module barrel (`flights/index.ts`) for cross-module consumption by declarations, united-declarations, and sorting.
- "Bağlama sayı" column header corrected from "Say" to match old module.
- "Başlanğıc tarixi" header corrected from "Başlama tarixi".
- Close-flight row action changed from single `Modal.confirm` + direct API call to a 3-item submenu navigating to the close modal, matching old behavior exactly.
